# Admin API for [erldns](https://github.com/dnsimple/erldns)

This app provides an admin API for querying and controlling an erldns server.

## Building

To build:

```bash
make
```

To start fresh:

```bash
make fresh
make
```

## Running

### To get an interactive Erlang REPL:

```bash
./rebar3 shell
```
